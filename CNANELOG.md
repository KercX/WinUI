# 📑 Changelog

All notable changes to the **WinUI** project will be documented in this file.

## - 2026-03-19
### Added
- New **Slider** element with smooth dragging.
- **Acrylic Blur** (Beta) for main window background.
- Professional **setup.bat** for developer automation.
- GitHub Actions **Workflow** for automated releases.
### Changed
- Improved **Ripple Effect** performance on low-end PCs.
- Refactored **Tab System** for better scaling.

### Fixed
- Fixed a bug where the window wouldn't drag on mobile devices.
- Fixed UI flickering when opening multiple tabs.

## - 2026-03-20
- Initial Release of WinUI.
